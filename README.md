Coding in Go Lang

## Notes

### Maps
 1. Create a map:
    ``` 
    myMap := make(map[int] int)
    ```
 2. check if map contains key:
    ```
     if val, ok := myMap[key]; ok {
            //do something
     }
    ```
 3. Add/Update map
    ```
    myMap[key] = val
    ```
